Pendant ce treimestre on rencontré plusieurs props CSS dont on va siter quelques une

Commençant par les balises à utiliser pour mettre en page et en forme un formulaire comme le TP3 Vous pouvez utiliser les propriétés CSS suivantes pour rendre ce champ de saisie plus attrayant.:

  • Padding

    • ajouter un espacement intérieur


  • Margin

    • pour ajouter une marge autour du champ de saisie


  • Border

    • pour ajouter une bordure aux champs de saisie


  • Box shadow

    • Vous pouvez utiliser la propriété CSS box-shadow pour ajouter une ombre portée. Vous pouvez obtenir une gamme d'effets en jouant avec les cinq valeurs de la propriété.


  • Border radius

    • La propriété border-radius peut avoir un impact considérable sur la sensation de vos formulaires. En incurvant les bords des cases, vous pouvez modifier considérablement l'apparence de vos champs de saisie.


  • Fonts

    • La plupart des navigateurs utilisent une famille et une taille de police différentes pour les éléments de formulaire. Si nécessaire, nous pouvons hériter de la police du document.
