le HTML c'est quoi?

  • 01 D'abord HTML abbréviation de ?

    Hyper Text Markup Language

  • 02 Définition

    HTML, désigne un type de langage informatique descriptif. Il s'agit plus précisément d'un format de données utilisé dans l'univers d'Internet pour la mise en forme des pages Web. Il permet, entre autres, d'écrire de l'hypertexte, mais aussi d'introduire des ressources multimédias dans un contenu.

  • 03Pourquoi HTML?

    HTML est répandu et est devenu plus expérimenté que jamais. Le développeur peut mettre en service des applications du concept HTML en temps réel.

 

Pendant ce treimestre on rencontré plusieurs balises HTML dont on va siter quelques une

Commençant par les balises à utiliser pour construire un formulaire comme le TP3 L'élément

peut contenir un ou plusieurs des éléments de formulaire suivants :
  • input

    • sert à faire une zone de saisie de données par l'utilisteur , il y'a plusieurs types comme les zones de textes ou les checkbox(s)


  • textarea

    • une zone de saisie de données de type texte comme le nom de l'utilisateur ou son addresse s


  • button

    • On utilise les boutons dans un formulaire pour envoyer les donner avec un bouton de type submit pour envoyer les données vers le serveur avec une requte php q'on va voir au cours de cette année ou pour choisir entre deux options comme le choix du genre de l'utilisateur en employant un boutton de type radio


  • select

    • on utilise select pour créer une liste déroulante


  • option

    • on l'utilise por ajouter des option à notre liste déroulante


  • optgroup

    • on utilise celle-là pour grouper les options de notre liste à des catégories par exemple les pays pour séparer les pays de différents continents

    Ensuite

    les balises à utiliser pour construire un tableau comme le TP1 et TP2 L'élément peut contenir un ou plusieurs des éléments de tab suivants :
  • tr

    • elle signifie "table row" ou ligne de tableau


  • td

    • elle correspond à "table data"ou les données d'une cellule du tableau


  • th

    • elle correspond à "table header" ou les cellules de l'en-tête de tableau.


  • Caption

    • ça sert à définir une légende de tableau


  • colgroup

    • Spécifie un groupe d'une ou plusieurs colonnes dans un tableau pour le formatage du tableau


  • tfooter

    • Regroupe le contenu du pied de page dans un tableau
